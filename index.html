About

Algo Amalgamation is a website dedicated to teaching novice programmers and introductory Computer Science students fundamental searching and sorting algorithms.

What is an algorithm?

Generally speaking, an algorithm is a set of defined rules for solving some sort of problem. Perhaps the algorithm involves rearranging a list of numbers in sorted order or calculating the shortest path to a destination. Whether we realize it or not, we all use algorithms in our daily lives to guide our actions and have been doing so since a very early age.

Why Study Algorithms?

A basic understanding of data structures and algorithms are essential for doing serious work in any branch of computer science. The most famous example would be the page ranking algorithm used by Google's search engine.
Searching and sorting are ubiquitous in practical algorithms and segue into a variety of topics, including divide-and-conquer, stability, and computational complexity. It is for this reason that they are among the first algorithms taught, and they will be the focus of this website.
